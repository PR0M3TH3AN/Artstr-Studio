Disc artwork

Avery 8960 uses two 4.625 in full-face disc labels on Letter paper. The center holes are kept clear in the preview and PDF.

Layers
Selected layer

Click a layer on the preview to select it. Drag selected image or text layers directly on the page. Text layers are editable in place.

Deployment notes

Save this file as index.html and host it anywhere static: Netlify, GitHub Pages, S3, Nginx, or just open it locally. Images stay in your browser unless you save a project JSON.
