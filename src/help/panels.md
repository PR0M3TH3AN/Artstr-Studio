# Sidebar panels — overview

Artstr Studio has two stacks of panels around the canvas:

- **Left sidebar** — persistent. Holds project metadata, the layout
  picker, mode-specific settings, the layers list, and the export +
  save/print buttons.
- **Right pane** (the floating **Tool Options**) — contextual.
  Changes based on the active tool + the selected layer.

## Left-sidebar groups

| Panel | What it covers |
|-------|----------------|
| [Project Metadata](#/help/panel-metadata) | Title, category, IDs, language, custom tag. |
| Template type picker | Switches between Cover / Disc / Jewel / Custom Art / Slide / Deck / Pixel Art / Book — see the per-mode pages under *Template-mode layouts*. |
| Mode-specific fieldsets | Cover size + artwork mode, Custom Art dimensions, Deck theme + import, Book document settings, etc. (Each mode's page covers its panel.) |
| [Credits](#/help/panel-credits) | Linked-design contributors surfaced for attribution. |
| [Layers](#/help/panel-layers) | The per-mode layer list. Where most of your interaction lives. |
| [Export options](#/help/panel-export) | Show / hide / print guides + bleed + crops. |
| [Save / Print](#/help/panel-print-save) | Project JSON save/open + Print/Save as PDF + Export JPEG/PNG. |

## Right pane

- [Tool Options](#/help/panel-tool-options) — overview of the
  contextual pane and how it switches with the active tool.

The per-tool sections under *Vector tools* (Pen, Shape, Text, etc)
cover what each tool's contextual options do.
